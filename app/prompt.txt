На вход подается два поля: "ЗАДАЧА" и "ГЛОБАЛЬНАЯ ЦЕЛЬ". Каждое поле разграничено тройными открывающими кавычками.
Тебе нужно ответить соответствует ли приведенная задача концепции SMART.
Для каждого из пунктов тебе нужно ответить почему ты так считаешь в поле "reason", а в поле "conclusion" дать вердикт.

Критерий "R" рассматривай отдельно. Если задана "ГЛОБАЛЬНАЯ ЦЕЛЬ", то оценивать "R" нужно по нему.
Если же поле "ГЛОБАЛЬНАЯ ЦЕЛЬ" пустое, то поле R должно выглядеть так:
```
"R": {
            "reason": "",
            "conclusion": true,
        },
```

Пример ответа заключен в тройные открывающие кавычки. Поля "reason" должны быть на том же языке, что поле "ЗАДАЧА".
Используй следующий JSON формат для ответа:
```
{
    "S": {
            "reason": <text>,
            "conclusion": <bool>,
        },
    "M": {
            "reason": <text>,
            "conclusion": <bool>,
        },
    "A": {
            "reason": <text>,
            "conclusion": <bool>,
        },
    "R": {
            "reason": <text>,
            "conclusion": <bool>,
        },
    "T": {
            "reason": <text>,
            "conclusion": <bool>,
        },
}
```

```
{task}
```

ГЛОБАЛЬНАЯ ЦЕЛЬ:
```
{goal}
```
