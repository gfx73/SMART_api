На вход подается два поля: "ЗАДАЧА" и "ГЛОБАЛЬНАЯ ЦЕЛЬ". Каждое поле разграничено тройными открывающими кавычками.

Мы будем говорить о соответствии задачи концепции SMART.

1) Для каждого из критериев SMART тебе нужно ответить почему ты так считаешь в поле "reason", а в поле "conclusion" дать вердикт.
При этом критерий "R" рассматривай отдельно. Если задана "ГЛОБАЛЬНАЯ ЦЕЛЬ", то оценивать "R" нужно относительно нее.
Если же поле "ГЛОБАЛЬНАЯ ЦЕЛЬ" пустое, то для критерия R "reason": "", "conclusion": true.

2) В конце нужно предложить улучшенный вариант задачи, который соблюдает принципы SMART.

Пример ответа заключен в тройные открывающие кавычки. Поля "reason" должны быть на том же языке, что поле "ЗАДАЧА".
Используй следующий JSON формат для ответа:
```
{
    "S": {
            "reason": <text>,
            "conclusion": <bool>,
        },
    "M": {
            "reason": <text>,
            "conclusion": <bool>,
        },
    "A": {
            "reason": <text>,
            "conclusion": <bool>,
        },
    "R": {
            "reason": <text>,
            "conclusion": <bool>,
        },
    "T": {
            "reason": <text>,
            "conclusion": <bool>,
        },
    "enhanced": <text>,
}
```

```
{task}
```

ГЛОБАЛЬНАЯ ЦЕЛЬ:
```
{goal}
```
